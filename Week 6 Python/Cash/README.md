# 💰 Cash (Python)

## 📄 File
`cash.py`

## 📝 Description
Implements a greedy algorithm to return the minimum number of coins for change.

## 🚀 How to Run
```bash
python cash.py
```

## 🧠 Concepts
- Python control structures
- String and file manipulation
- Cleaner syntax compared to C
- No manual memory management
