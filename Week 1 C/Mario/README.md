# 🧱 Mario (Less)

## 📄 File
`mario.c`

## 📝 Description
Prints a pyramid of hashes `#` of a given height using nested loops.

## 🚀 How to Run
```bash
gcc -o mario mario.c
./mario
```

## 🧠 Concepts
- Compilation with `gcc`
- Loops and conditionals
- User input/output
- Basic C syntax
