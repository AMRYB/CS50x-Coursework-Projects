# 🔐 Caesar Cipher

## 📄 File
`caesar.c`

## 📝 Description
Encrypts plaintext using Caesar’s cipher by shifting characters with a provided key.

## 🚀 How to Run
```bash
gcc -o caesar caesar.c -lm
./caesar 2
```

## 🧠 Concepts
- Arrays and loops
- Character manipulation
- Command-line arguments
- Conditional logic
