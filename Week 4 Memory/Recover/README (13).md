# 🧼 Recover

## 📄 File
`recover.c`

## 📝 Description
Recovers JPEG images from a forensic memory card image (e.g., card.raw).

## 🚀 How to Run
```bash
gcc -o recover recover.c
./recover card.raw
```

## 🧠 Concepts
- Pointers and memory manipulation
- File I/O with binary data
- Structs and arrays
- Audio/image processing
