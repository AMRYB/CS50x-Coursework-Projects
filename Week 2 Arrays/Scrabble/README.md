# 🔤 Scrabble

## 📄 File
`scrabble.c`

## 📝 Description
Compares two words and scores them using Scrabble point values.

## 🚀 How to Run
```bash
gcc -o scrabble scrabble.c -lm
./scrabble
```

## 🧠 Concepts
- Arrays and loops
- Character manipulation
- Command-line arguments
- Conditional logic
