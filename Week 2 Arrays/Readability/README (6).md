# 📊 Readability

## 📄 File
`readability.c`

## 📝 Description
Analyzes text and returns the reading grade level using the Coleman-Liau index.

## 🚀 How to Run
```bash
gcc -o readability readability.c -lm
./readability
```

## 🧠 Concepts
- Arrays and loops
- Character manipulation
- Command-line arguments
- Conditional logic
