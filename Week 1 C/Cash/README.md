# 💰 Cash

## 📄 File
`cash.c`

## 📝 Description
Implements a greedy algorithm to determine the minimum number of coins needed to make change.

## 🚀 How to Run
```bash
gcc -o cash cash.c -lm
./cash
```

## 🧠 Concepts
- Compilation with `gcc`
- Loops and conditionals
- User input/output
- Basic C syntax
