# 👋 Hello, It's Me

## 📄 File
`hello.c`

## 📝 Description
Prints `hello, world` to the screen. A basic program to demonstrate syntax and compilation.

## 🚀 How to Run
```bash
gcc -o hello hello.c
./hello
```

## 🧠 Concepts
- Compilation with `gcc`
- Loops and conditionals
- User input/output
- Basic C syntax
