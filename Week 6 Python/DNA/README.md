# 🧬 DNA

## 📄 File
`dna.py`

## 📝 Description
Identifies a person based on DNA STR counts by matching against a CSV database.

## 🚀 How to Run
```bash
python dna.py
```

## 🧠 Concepts
- Python control structures
- String and file manipulation
- Cleaner syntax compared to C
- No manual memory management
